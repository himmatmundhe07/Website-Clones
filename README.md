* **Product Cards:** Detailed product listings with hover effects, pricing, and ratings.
* **Multi-page Navigation:** Most projects include linked sub-pages (e.g., "Contact Us", "Cart", "Story").
* **Forms:** Styled login forms, newsletter subscriptions, and contact inputs.

## üöÄ How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/himmatmundhe07/Website-Clones.git](https://github.com/himmatmundhe07/Website-Clones.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Website-Clones
    ```

3.  **Open the Dashboard:**
    Open the root `index.html` file in your preferred browser. This file acts as a table of contents linking to all the individual projects.
    
    *Alternatively, navigate into specific folders (e.g., `/Bombay-closet`) and open the `index.html` found there.*
